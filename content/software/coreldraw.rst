5 Ways to Copy an Object in Corel Draw 
######################################

:tags: coreldraw, tips
:date: 2015-05-26 17:00


- `num-plus` duplicate with zero distance [*]_

- `ctrl-d` duplicate with distance set in the toolbar [*]_

- `right-click` when dragging

- `space` when dragging

- `ctrl-r` repeat last action


.. [*] Non-remappable to other keys other than by creating a macros

.. [*] Starts 'duplication mode':Â  records copied movement and rotation and applies them (instead of the distance from the toolbar and zero rotation) on subsequent `ctrl-d` strokes until another object is selected (resetting distance and rotation to original).
